# 🚀 Revadi Naga Sai Jaswanth — 3D Developer Portfolio (React + TypeScript + Three.js)

A modern, high-performance **3D developer portfolio website** built with **React**, **TypeScript**, **Three.js**, **GSAP**, and **WebGL**.

---

## ✨ Highlights

- **3D / WebGL experience** powered by **Three.js**
- Smooth interactive animations with **GSAP** & **ScrollTrigger**
- Modern **React + TypeScript** modular codebase
- Clean dark-mode UI & full responsive design (desktop + mobile)
- Built-in sections for **About**, **Professional Experience**, **Education**, **Projects**, **Certifications**, and **Contact** with direct email integration

---

## 🧰 Tech Stack

- **Frontend**: React, TypeScript, HTML5, CSS3, JavaScript
- **3D & Animation**: Three.js, WebGL, GSAP, ScrollTrigger, Lenis Smooth Scroll
- **Icons & UI**: React Icons, Recharts
- **Build Tool**: Vite

---

## 🚀 Getting Started

### 1) Clone

```bash
git clone https://github.com/Revadi-Jaswanth/jaswanth_revadi-portfolio.git
cd jaswanth_revadi-portfolio
```

### 2) Install Dependencies

```bash
npm install
```

### 3) Run Locally

```bash
npm run dev
```

### 4) Build for Production

```bash
npm run build
```
